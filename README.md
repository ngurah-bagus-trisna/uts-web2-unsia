# How to install

1. Clone repository

2. Buat file `.env` dengan format

```
DB_NAME="db_inventory"
DB_USERNAME="root"
DB_PASSWORD="<password>"
DB_HOST="<host-mysql-ip>"   
```

3. Compose UP!

```sh
docker compose up -d
```
